# tia-tree-calls

Read a block call graph (generated by [tia-tree-calls](../tia-tree-calls/)) and display a visual tree of the call hierarchy.

## Usage

```bash
tia-tree-calls [flags]
```

Read `block_calls.json` and show all top-level blocks recursively:

```bash
tia-tree-calls
```

Show call tree starting from a specific entry:

```bash
tia-tree-calls -e OB_Program_Cycle
```

Show only direct children (depth 1):

```bash
tia-tree-calls -e FB_Bin_Control -l 1
```

Filter by block type:

```bash
tia-tree-calls -t FB,FC
```

Write output to a file:

```bash
tia-tree-calls -i custom.json -o tree.txt
```

## Flags

| Short | Long            | Default            | Description                                                      |
|-------|-----------------|--------------------|------------------------------------------------------------------|
| `-i`  | `--input-file`  | `block_calls.json` | Path to the JSON call graph file                                 |
| `-e`  | `--entry`       |                    | Comma-separated entry block names. Omit to show top-level blocks |
| `-o`  | `--output-file` |                    | File path to write output. Omit to print to console              |
| `-t`  | `--type`        |                    | Comma-separated block types to filter (e.g. FB,FC)               |
| `-l`  | `--level`       | `0`                | Maximum expansion depth. `0` = unlimited                         |
| `-v`  | `--version`     |                    | Show version                                                     |
| `-h`  | `--help`        |                    | Show this help message                                           |

## Example output

```text
OB_Main
└── FB_Init
    └── FC_Util
```

Cyclic references are detected and stop further expansion of that branch.

## Version

```bash
tia-tree-calls -v
```

The version is injected at build time:

```bash
go build -ldflags="-X main.version=1.0.0" ./tia-tree-calls
```
